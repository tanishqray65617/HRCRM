# HRCRM Project

A full-stack Human Resource Customer Relationship Management application.

## Project Structure
- `frontend/`: React + Vite application with TailwindCSS for the user interface.
- `backend/`: Node.js + Express server, utilizing Supabase for database and authentication.

## Running Locally

1. **Start the backend server:**
   Navigate to the `backend` directory and run:
   ```bash
   npm run dev
   ```
   The backend will be available at `http://localhost:5000`.

2. **Start the frontend application:**
   Navigate to the `frontend` directory and run:
   ```bash
   npm run dev
   ```
   The frontend will be available at `http://localhost:5173`.
